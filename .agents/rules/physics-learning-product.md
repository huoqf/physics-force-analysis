# Physics Learning Product Rules

本项目是高中物理受力分析学习辅助工具，不是单纯的物理动画演示。

## 产品目标
1. 引导学生选择研究对象。
2. 引导学生判断真实受力。
3. 引导学生画出受力图。
4. 对错误判断给出物理原因解释。

## 设计原则
- 不要直接给出完整答案，优先采用分步提示。
- 每个力都必须说明施力物体、受力物体、方向和存在原因。
- 所有反馈都要符合高中物理表述，不能出现“下滑力”等错误概念。

## 技术原则
- 使用 Vue 3 Composition API。
- 物理场景数据与 UI 组件分离（数据驱动）。
- 交互流程由数据中的 `steps` 驱动，错误反馈由 `commonMistakes` 驱动。

## 环境配置
- 项目开发与运行依赖于安装在 `D:\node-v24` 的便携版 Node.js。
- 执行 `npm` 或 `node` 命令时，需确保该目录已添加至系统 PATH，或在执行脚本中显式指定该路径。
